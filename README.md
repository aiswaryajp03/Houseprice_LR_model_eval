# Houseprice_LR_model_eval

-> Data Pre-processing: Encodes categorical variables, applies one‑hot encoding for furnishing status, and handles boolean indicators (mainroad, guestroom, basement, etc.).

->Exploratory Data Analysis: Provides insights through statistical summaries, box plots, histograms, and a correlation heatmap.

-> Model Training and Evaluation:
Trains a Linear Regression model on the dataset.
Evaluates performance using MAE, MSE, and R² metrics.
Visualizes results with an actual vs. predicted price scatter plot.
